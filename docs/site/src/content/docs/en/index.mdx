1. **[Quick start](/docs/en/getting-started/)** — map your scenario to the [examples](https://github.com/bugkiwi/OtterVoice/tree/main/examples) in the repo.
2. **[Package overview](/docs/en/packages/)** — which packages to install and what each owns.
3. **Per-package guides** — options, events, pitfalls. Source JSDoc and the [API reference](/docs/en/reference/) are authoritative for signatures.
